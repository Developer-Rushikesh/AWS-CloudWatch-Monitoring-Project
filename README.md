Project Title

AWS CloudWatch Monitoring and Alerting System

Project Overview
This project demonstrates real-time monitoring and alerting using AWS services.

The project includes:

EC2 server setup
Website hosting using Nginx
CloudWatch monitoring
CPU utilization alarms
SNS email notifications
Real-time server performance tracking

This project helps beginners understand AWS monitoring and cloud operations practically.

Project Architecture

EC2 Instance → CloudWatch Metrics → Alarm → SNS → Email Notification


Features
Real-time CPU monitoring
Automatic email alerts
Website hosting on EC2
Linux server management
Cloud monitoring dashboard
Performance testing using stress tool


Steps Performed
1. Launch EC2 Instance
Created Ubuntu EC2 instance
Allowed SSH and HTTP ports
Connected using SSH key pair
2. Install Nginx
Installed Nginx web server
Hosted sample webpage
Verified website using public IP
3. Configure CloudWatch
Opened EC2 metrics
Monitored CPU utilization
Viewed real-time graphs
4. Create CloudWatch Alarm
Created CPU utilization alarm
Set threshold value
Configured alarm state actions
5. Configure SNS Notifications
Created SNS topic
Added email subscription
Confirmed email subscription
6. Test Alarm
Installed stress tool
Generated CPU load
Verified alarm trigger and email notification
